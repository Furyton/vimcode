## interesting shortcut

- `Alt+j`/`Alt+k` moves current line down/up.
- `Ctrl+/` comment current line, only non-blank line.
- use `jk` to escape.
- `<Leader> f` / `<Leader> b` fuzzy find in files / buffers.
